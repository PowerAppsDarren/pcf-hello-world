

## Package & Deploy

```bash
# Make a new subfolder in your project
mkdir Solution
# Move into that new folder
cd Solution
# Now create your new solution there
pac solution init --publisher-name "PUBLISHER_NAME" --publisher-prefix "YOUR_PREFIX"
```
